## Pardus 23.4 GNOME için – GTK4 + Libadwaita Uygulaması  Bu uygulama GNOME menüsünde “Öğretmen Paylaşımı Kurucu (GNOME)” diye görünecek. Modern Adwaita temalı, sade bir pencere olacak.

## GNOME tarafında:
### sudo apt install python3-gi gir1.2-gtk-4.0 gir1.2-adw-1 cifs-utils gvfs-backends gvfs-fuse

## İzin ver
### sudo chmod +x ogretmen_paylasim_gnome.py
## Çalıştır
## sudo python3 ogretmen_paylasim_gnome.py
